Implementation of Prefetch guard in L2 cache

Dec 2023
The project aims to arm the prefetcher of a core with capabilities to detect suspicious side channel attack and help mitigate such attacks. we firstly design a method to determine whether the misses in cache are conflict misses, then we detect a pattern in consecutive miss sets and finally devise and test a prefetcher controller that prefetches data intelligently to obfuscate the spy's probe. This would be implemented the lower most cache - L2 cache, of a Rocket core.

Individual Role:

  • Implemented a prefetcher in RISC V Rocket Core for side channel attack mitigation.
  • Modified Arbiter and other Rocket core blocks to allow for additional buffers in cache that can help detect attacks
  • Achieved a 70% attack detection in L2 cache memory, Skills gained: Computer architecture, prefetcher implementation, Scala programming, Chisel Framwork, RISC-V
